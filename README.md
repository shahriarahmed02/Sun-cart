# SunCart – Summer Essentials Store ☀️

SunCart is a modern e-commerce platform built for the **SunCart Summer Essentials Assignment**. It features a curated list of summer products, secure authentication, and a sleek user interface.

## 🚀 Live Demo
[https://sun-cart-pearl.vercel.app/]

## ✨ Key Features
- **Responsive Home Page**: Beautiful hero section with Animate.css and featured products.
- **Authentication**: Secure Login/Register using **BetterAuth** (Google & Email/Password).
- **Protected Routes**: Product details are only accessible to logged-in users.
- **Dynamic Product Loading**: Fetching data from a local JSON file.
- **Profile Management**: Users can view and update their profile information.

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & DaisyUI
- **Auth**: BetterAuth
- **Animations**: Animate.css & Lottie
- **Icons/Toasts**: React Hot Toast

## 📦 Installation
1. Clone the repo: `git clone https://github.com/shahriarahmed02/SunCart`
2. Install dependencies: `npm install`
3. Set up `.env.local` with your Google Client ID and Secret.
4. Run locally: `npm run dev`
